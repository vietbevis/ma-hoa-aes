🔄 Round-trip Tests TÍNH THUẬN NGHỊCH

Kiểm tra tính thuận nghịch: mã hóa rồi giải mã phải cho lại kết quả gốc chính xác. Bao gồm:

  • Text ASCII ngắn
  • Text tiếng Việt (UTF-8 multi-byte)
  • Dữ liệu nhị phân 1024 bytes
  • AES-128, AES-192, AES-256
🔧 Utility Tests CÁC HÀM HỖ TRỢ

Kiểm tra Base64, Hex encoding, UTF-8 encode/decode, PKCS#7 padding — tất cả tự cài đặt.

🛡 Security Properties TÍNH AN TOÀN

Kiểm tra tính chất bảo mật của AES:

  • Avalanche Effect: Thay đổi 1 bit khóa → ciphertext thay đổi >40%
  • Key Sensitivity: Sai khóa → không giải mã được
